# 🚨 JPFA 緊急サイト バックアップ

**一般社団法人 日本電動車椅子サッカー協会** の緊急時暫定サイトです。

## ⚡ GitHub Pages で即座に確認

このリポジトリをGitHub Pagesで公開するには：

1. **Settings** タブをクリック
2. 左メニューから **Pages** を選択  
3. **Source** で **Deploy from a branch** を選択
4. **Branch** で **main** を選択
5. **Save** をクリック

数分後、以下のURLでサイトが公開されます：
`https://alohaloha.github.io/jpfa_test/`
